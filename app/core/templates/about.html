About

This software is designed to track stats through snapshots, providing users with the ability to add and customize UI elements on a grid. These elements can be moved around freely, allowing for a personalized layout.

Users can connect events to their code, enabling dynamic UI updates when specific events are triggered. Originally, this application was developed to track Overwatch 2 (OW2) games, supporting both keyboard and controller inputs.

Features

  • Customizable UI elements on a grid layout
  • Event-driven UI updates
  • Support for keyboard and controller inputs
